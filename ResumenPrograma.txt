Esquema Programa Machine Learning

Descarga de los datos, por meses, se realiza la media horaria por cada estación y se añade la información de si ese día llueve (el observatorio de Fabra detecta lluvia), o si hace viento fuerte (mas de 20m/s) ese día.
A parte se añaden la información de Dia del año, mes, hora, día de la semana, …
Se descartan todas esas líneas que contengan nans o problemas.

Programas de análisis:
Para los programas de análisis se prepara también el archivo para hacer las predicciones
Se utilizan para entrenar solo los datos de 2021 y 2022 para no tener datos del covid.
A demás se añade la información de si es día de fiesta (sábado o domingo o agosto) y la estación del año.
Se descartan todas esas líneas que contengan nans o problemas.

Red Neuronal: 
- Al pirncipio la red neuronal era demasiado pequeña y el programa no conseguía poder claisficar y obtenia como solución la media de ocupación de las estaciones, siempre precedia lo mismo.
- Se augmenta las capas de red neuronal, y la cantidad de neuronas, con capas dense y también capas para eliminar oytlayers. De esta forma se consigue un mejor resultado però todavía lejos de lograr un resultado aceptable.

Lightgbm:
- Se utiliza este programa con los mismos datos i valores que la red neuronal, pero es mucho más rápido y además permite decir que variables son categóricas y cuales no. Esto mejora el un poco más el cálculo porque el ID de las estaciones es categórico. Pero en la red neuronal no podíamos hacer un onehotencode porqué son demasadas estaciones.
